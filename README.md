# Installation & Setup

## Clone the repo
```
git clone https://github.com/AlinaPathan/Softsell-Marketing-Site.git
```
## Go into the project
```
cd Softsell
```
## Install dependencies
```
npm install

npm install lucide-react
```

## Run the dev server
```
npm run dev

```
Make sure you have Node.js and npm installed.
